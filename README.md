# Meeting Transcriber

Консольное приложение для локальной транскрибации совещаний. Захватывает системный звук и микрофон, нарезает по паузам, транскрибирует и пишет в JSON с таймкодами.

Работает полностью офлайн.

## Что нужно

- Windows 10/11
- Python 3.10+
- NVIDIA GPU (8+ GB VRAM) — для Whisper. GigaAM e2e работает и на CPU
- Модели скачиваются автоматически при первом запуске

## Установка

Открыть терминал (Win+R → ввести `cmd` → Enter) и выполнить по очереди:

**1. Скачать проект**
```bash
git clone https://github.com/HRYNdev/transcriber
cd transcriber
```

**2. Установить зависимости**
```bash
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

Запускать до начала встречи. Приложение захватывает системный звук и микрофон одновременно.

Остановить — **Ctrl+C**. Результат появится в папке `meetings/`.

Файл называется по дате и времени: `2026-03-31_08-19-47.json`

## Настройки

Все параметры в файле `config.yaml`. После изменений перезапустить приложение.

### Выбор модели

```yaml
model:
  type: "gigaam_e2e"   # gigaam_e2e, gigaam или whisper

  gigaam_e2e:
    variant: "e2e_ctc"   # e2e_ctc (быстрее) или e2e_rnnt (точнее пунктуация)
```

Доступные модели:

| Модель | Пунктуация | GPU | Описание |
|--------|-----------|-----|----------|
| `gigaam_e2e` | Да | Не обязателен | GigaAM v3 с нормализацией текста |
| `whisper` | Да | Рекомендуется | OpenAI Whisper large-v3 |
| `gigaam` | Нет | Не обязателен | GigaAM v3 (CTC/RNNT без нормализации) |

### Чувствительность VAD

```yaml
vad:
  threshold: 0.5          # 0.1–0.9, ниже = чувствительнее

audio:
  silence_duration: 1.5   # секунд тишины для завершения сегмента
```
