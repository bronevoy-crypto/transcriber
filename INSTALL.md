# Установка и запуск

## Требования

- Windows 10/11
- Python 3.10+
- NVIDIA GPU с поддержкой CUDA (рекомендуется 8+ GB VRAM)
- 4+ GB свободного места для моделей

## Установка

### 1. Клонировать репозиторий

```
git clone <ссылка на репо>
cd Transcriber
```

### 2. Установить зависимости

```
pip install -r requirements.txt
```

### 3. Скачать модель GigaAM

```
python download_models.py
```

Скачается модель GigaAM v3 CTC (~500 MB) в папку `models/gigaam-v3/`.

> Если нужна более точная RNNT модель (WER ~8.3% против 9.1%):
> ```
> python download_models.py --rnnt
> ```
> После скачивания в `config.yaml` изменить `gigaam.type: "transducer"`

## Запуск

```
python main.py
```

Запустить **до** начала конференции. Приложение автоматически захватывает системный звук через WASAPI loopback — VB-Cable не требуется.

Остановить: **Ctrl+C**

Результат сохраняется в папку `meetings/` в формате JSON.

## Формат результата

Файл называется по дате и времени начала: `2024-01-15_10-30-00.json`

```json
{
  "meeting_date": "2024-01-15T10:30:00",
  "segments": [
    {
      "start": 5.48,
      "end": 12.30,
      "speaker": "SPEAKER_00",
      "text": "Добрый день, начинаем совещание"
    }
  ]
}
```

## Смена модели

В файле `config.yaml`:

```yaml
model:
  type: "gigaam"   # gigaam или whisper
```

Для переключения на Whisper:
1. Изменить `type: "whisper"`
2. Перезапустить приложение

## Настройка VAD

В `config.yaml` раздел `vad`:

- `threshold` — чувствительность (0.1–0.9, по умолчанию 0.5). Снизить если речь не подхватывается, повысить если много шума.
- `silence_duration` — секунд тишины для завершения сегмента (по умолчанию 1.5).
